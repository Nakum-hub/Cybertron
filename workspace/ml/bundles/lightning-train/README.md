# Lightning AI H100 Training Bundle

This bundle is the shortest path from local prep to a paid Lightning AI GPU run.

## What to upload

Upload the entire prepared folder generated by:

```powershell
npm run ml:prepare:lightning --prefix workspace
```

The bundle contains only:

- `train_cybertron_lora.py`
- `requirements.txt`
- `data/cybertron_training_corpus.jsonl`
- `run_lightning_train.sh`
- `README.md`
- `bundle_manifest.json`

## Recommended workflow for a paid H100 hour

Do **not** spend H100 time on data prep.

Do all of this first on the local machine:

```powershell
npm run ml:export:bootstrap
$env:GITHUB_TOKEN = gh auth token
$env:GITHUB_MODELS_MODEL = "openai/gpt-4.1-mini"
npm run ml:github-models:teacher -- --variants 3
npm run ml:build:corpus
npm run ml:prepare:lightning
```

Then on Lightning:

1. Start the Studio with the H100 machine only when the bundle is ready.
2. Upload the prepared bundle folder or move it into the workspace.
3. Open a terminal in the Studio.
4. Run:

```bash
cd lightning-train
bash run_lightning_train.sh
```

## Storage truth

- The adapter is written to `outputs/bootstrap-risk-policy-threat`
- A tarball is created at `outputs/bootstrap-risk-policy-threat.tar.gz`
- Do **not** trust remote workspace storage as the only copy
- Always download the tarball after training

## Should you use H100 for this exact dataset?

Only if you care about fast turnaround more than efficiency.

This corpus is still small relative to real foundation-model training. An H100 will run it easily, but it will not magically create a production-grade Cybertron model. The best use of the H100 hour is:

1. run one strong multi-task adapter job
2. save the adapter
3. if time remains, run one second job with a different base model or longer epoch count

The default bundle now assumes a stronger base model and fuller settings:

- `Qwen/Qwen2.5-7B-Instruct`
- `NUM_TRAIN_EPOCHS=6`
- `LORA_RANK=32`
- `GRADIENT_ACCUMULATION_STEPS=16`

Only if the machine price still fits and the first run path is proven should you try something larger. Example:

```bash
export MODEL_NAME="Qwen/Qwen2.5-14B-Instruct"
```

Only do that after the first run path is proven, and only on a single H100. Do not use 2x H100 for this dataset.
