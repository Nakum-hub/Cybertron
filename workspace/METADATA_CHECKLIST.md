# Metadata Checklist

For every new page or product module:
- [ ] Unique `<title>` with brand + intent.
- [ ] Unique meta description (benefit-oriented).
- [ ] Canonical URL defined by deploy layer.
- [ ] Open Graph title/description/image.
- [ ] Route included in sitemap generation.
- [ ] Robots policy confirmed for target environment.
- [ ] Structured data considered where relevant.
